# T-YEP-600

## Installation

### Backend

Install Nest.js

```sh
sudo npm install -g @nestjs/cli
nest --version
```

Install YARN

```sh
sudo npm install yarn -g 
```

Update Node.js to the last version

```
sudo npm install -g n
sudo n lts
sudo n prune
```

Install dependancies

```sh
cd backend
npm install
```

### Frontend

Install Node modules

```sh
cd backend
npm install
```

Start on expo

```sh
npm run start
```
